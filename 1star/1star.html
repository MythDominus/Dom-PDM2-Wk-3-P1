Instructions

This exercise will walk you through the steps of creating an additional JS file and linking it in the HTML.

  1. Select the sketch folder ("1star") or any of the individual files in the sketch folder (but not one of the nested folders).
  2. Click the new file icon in the Explorer pane. It will be the first button that appears just to the right of the practical name when you move your mouse near it.
  3. Name the new file ball.js. Don't forget the extension ".js" otherwise it won't be recognised as a JavaScript file.
  4. In ball.js, write a class to represent a ball. At a minimum, it should have instance variables for a ball's x and y coordinates, and a method that draws the ball (just the ball, not a background). Eventually, you will use your class to create an animation of your choosing. An example is shown below but you may do something different. Feel free to add more instance variables and methods depending on what you would like to use the class for.
  5. Next, look at the code for this file, 1star.html. Scroll all the way down until you see the <script> tag that links sketch.js. Above or below the <script> tag, add another <script> tag that links the new ball.js file: 
    <script src="ball.js"></script>
  6. In sketch.js, write a sketch that uses the class in the other file in some way (see video for inspiration). You should be able to just use it as if it was defined in sketch.js. If you see an error message that says your class is not defined, try swapping the script tags.

Example output:

If you would like to create an effect similar to what is shown in the video, where the balls move away from the mouse, you will find the p5.js pmouseX and pmouseY system variables useful. The effect is created by using the current and previous values of the mouse coordinates to work out if the mouse is travelling in a positive or negative direction on each axis, then moving the ball in the same direction so it looks like it's moving away.
